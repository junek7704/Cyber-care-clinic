      🛡️ Breach Prevention Advice

      • Use a unique password for each account.
      • Enable two-factor authentication (2FA) wherever possible.
      • Change passwords immediately if you suspect a breach.
      • Avoid using personal information (birthdays, names, etc.).
      • Regularly check if your email/password has been exposed in a breach.

      📖 Password Best Practices

      🔄 How often should I change my passwords?
      Change important account passwords (like email, banking, and work accounts) every 3–6 months, or immediately if you suspect a breach.
      ❓ Why is password reuse dangerous?
      If one reused password is leaked in a data breach, attackers can try the same password on your other accounts (this is called credential stuffing).
      🔑 What makes a password truly strong?
      A strong password is long (12+ characters), contains a mix of upper/lowercase letters, numbers, and symbols, and avoids dictionary words or personal info.
      📱 Should I use a password manager?
      Yes! A password manager securely stores and generates unique passwords for each account so you don’t have to remember them all.
